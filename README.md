# 🎮 五子棋遊戲合集

一個包含多種遊戲模式的五子棋遊戲合集，支援AI對戰、本地對戰和網路對戰。

## 🌟 功能特色

- 🤖 **智能AI對戰** - 三種難度等級（簡單、中等、困難）
- 👥 **本地雙人對戰** - 在同一設備上進行對戰
- 🌐 **網路多人對戰** - 使用WebSocket實現即時對戰
- 📱 **響應式設計** - 完美支援手機和桌面設備
- 🎨 **精美界面** - 仿木質棋盤，立體棋子設計
- ⚡ **流暢體驗** - 優化的動畫效果和操作反饋

## 🎯 遊戲版本

### 1. gomoku_final.html - 完整版AI對戰
**主要特色：**
- 🤖 三種AI難度等級
- 🧠 先進的AI算法（威脅識別、戰略分析）
- 🎮 完整的遊戲控制功能
- 📱 完美的響應式設計

### 2. gomoku.html - 網路對戰版
**主要特色：**
- 🌐 WebSocket即時對戰
- 🏠 房間系統
- 👥 多人連線管理
- 💬 即時遊戲狀態同步

### 3. gomoku_basic.html - 基礎版本
**主要特色：**
- 👥 簡單的本地雙人對戰
- 🎯 標準五子棋規則
- 🔄 基本功能實現

## 🚀 快速開始

### AI對戰模式（推薦）
1. 打開 `gomoku_final.html`
2. 選擇「🤖 vs 電腦」
3. 選擇AI難度：
   - 😊 **簡單** - 適合初學者
   - 🤔 **中等** - 有一定挑戰性
   - 😤 **困難** - 高級AI，具備戰略思維

### 本地對戰模式
1. 打開任一HTML文件
2. 選擇「👥 本地對戰」
3. 黑棋先手，輪流下棋

### 網路對戰模式
1. 安裝Node.js依賴：`npm install`
2. 啟動伺服器：`npm start`
3. 打開 `gomoku.html`
4. 創建或加入房間開始對戰

## 🎯 遊戲規則

1. **標準五子棋規則**
   - 15×15棋盤
   - 黑棋先手
   - 首先連成五子者獲勝

2. **獲勝條件**
   - 橫向、縱向或斜向連成五子
   - 支援所有方向的五連判定

3. **操作說明**
   - 點擊空位下棋
   - 重新開始遊戲
   - 返回模式選擇

## 🛠️ 技術架構

### 前端技術
- **HTML5** - 語義化結構

- **威脅識別算法** - 識別活四、活三、雙三等模式
- **位置評估函數** - 綜合考慮攻防價值
- **戰略分析** - 多方向影響評估
